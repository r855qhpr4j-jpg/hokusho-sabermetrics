取込テンプレートの使い方

このページから、打者用と投手用のCSVテンプレートを確認できます。CSVはそのままExcelで開いて使えます。

打者テンプレートを開く 投手テンプレートを開く

使い方

  • CSVをExcelで開く
  • 1行目の見出しは消さずに、2行目以降へ各試合の成績を追加する
  • 選手名はサイトのプルダウンと同じ表記にそろえる
  • 保存するときはCSVのままで大丈夫です

打者テンプレートの列

  • date: 試合日 例 2026-04-01
  • opponent: 相手校
  • competition: 大会名や練習試合
  • player: 選手名
  • pa: 打席
  • single: 単打
  • double: 二塁打
  • triple: 三塁打
  • homeRun: 本塁打
  • runs: 得点
  • rbi: 打点
  • so: 三振
  • bb: 四球
  • hbp: 死球
  • sh: 犠打
  • sf: 犠飛
  • sb: 盗塁
  • cs: 盗塁死
  • gdp: 併殺打
  • note: メモ

投手テンプレートの列

  • date: 試合日 例 2026-04-01
  • opponent: 相手校
  • competition: 大会名や練習試合
  • player: 選手名
  • innings: 投球回 例 7.0 6.1
  • hitsAllowed: 被安打
  • hrAllowed: 被本塁打
  • strikeouts: 奪三振
  • walks: 与四球
  • hitByPitch: 与死球
  • runsAllowed: 失点
  • earnedRuns: 自責点
  • result: W L S など
  • starter: 先発なら TRUE
  • qualityStart: QSなら TRUE
  • completeGame: 完投なら TRUE
  • note: メモ
